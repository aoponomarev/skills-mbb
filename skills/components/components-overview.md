# components-overview

> Источник: `docs/doc-components.md` (раздел "Компоненты приложения")

## Общее правило

Детальный API каждого компонента (props, emits, slots, ref API) описан в шапках файлов в `shared/components/`.

## Базовые переиспользуемые компоненты (shared)

- `cmp-dropdown` → `shared/components/dropdown.js`, шаблон `shared/templates/dropdown-template.js`
- `cmp-combobox` → `shared/components/combobox.js`, шаблон `shared/templates/combobox-template.js`
- `cmp-dropdown-menu-item` → `shared/components/dropdown-menu-item.js`, шаблон `shared/templates/dropdown-menu-item-template.js`
- `cmp-button` → `shared/components/button.js`, шаблон `shared/templates/button-template.js`
- `cmp-button-group` → `shared/components/button-group.js`, шаблон `shared/templates/button-group-template.js`
- `cmp-modal` → `shared/components/modal.js`, шаблон `shared/templates/modal-template.js`
- `cmp-modal-buttons` → `shared/components/modal-buttons.js`, шаблон `shared/templates/modal-buttons-template.js`
- `cmp-timezone-selector` → `shared/components/timezone-selector.js`, шаблон `shared/templates/timezone-selector-template.js`
